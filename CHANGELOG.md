# Changelog

## v0.2.1 - 2026-06-09

### Changed

- 将扩展名称、工具栏标题、弹窗和上架文案统一更新为“公众号边听边读”。
- 更新发布包版本号到 0.2.1。

## v0.2.0 - 2026-06-09

### Fixed

- 停止正在朗读的文章 tab 关闭后残留的 TTS 播放，并在停止前保存断点进度。（#1）
- 提高关闭文章 tab 后阅读进度恢复的可靠性，增加页面生命周期兜底保存。（#2）
- 修复多个公众号文章 tab 同时操作时的播放状态串扰，改为单播放器显式接管。（#3）
- 修复 MV3 service worker 闲置后点击“继续”导致底部播放器偶发消失的问题。（#4）
- 简化“从这里读”为普通文章播放起点，并修复接管后 UI 显示播放但实际无声的问题。（#5）
- 修复微信文本页听读绿条错位、标题误入正文和扩展热更新后的旧脚本残留报错。（#6）

## v0.1.0

- 初始 Chrome 扩展版本，支持微信公众号文章本地语音听读、逐句高亮、自动滚动、语速调节和断点续读。
